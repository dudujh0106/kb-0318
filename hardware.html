🖥️ 하드웨어

CPU (Central Processing Unit)
컴퓨터의 연산과 처리를 담당하는 핵심 제어 장치.
Memory (RAM)
데이터를 임시 저장하는 장치로, 전원이 꺼지면 내용이 사라짐 (휘발성).
Disk (HDD/SSD)
데이터를 영구 저장하는 장치로, 전원이 꺼져도 데이터 유지됨 (비휘발성).
입출력장치 (I/O Device)
키보드, 마우스, 모니터, 프린터 등 사용자와 상호작용하는 장치.
GPU (Graphics Processing Unit)
컴퓨터 그래픽스 처리를 전담하는 프로세서로, 영상 및 이미지 렌더링에 사용됨.
Motherboard (메인보드)
컴퓨터의 모든 부품을 연결하는 중심 회로 기판으로, CPU, 메모리, 저장 장치 등이 장착됨.
Power Supply Unit (PSU)
컴퓨터에 전원을 공급하는 장치로, 전압을 변환하여 각 부품에 안정적인 전력을 제공함.
Cooling System (쿨링 시스템)
컴퓨터 부품의 온도를 조절하기 위해 팬, 히트싱크 등을 활용하여 열을 방출하는 장치.
Optical Drive (광학 드라이브)
CD, DVD, Blu-ray 디스크 등을 읽고 쓸 수 있는 장치로, 데이터 저장 및 재생에 사용됨.
Network Interface Card (NIC)
컴퓨터를 네트워크에 연결하여 데이터 송수신을 가능하게 하는 하드웨어 구성 요소.
Sound Card (사운드 카드)
컴퓨터의 오디오 처리를 담당하는 장치로, 소리의 입력과 출력을 관리함.
USB Port (USB 포트)
외부 장치를 컴퓨터에 연결하기 위한 포트로, 다양한 USB 장치를 지원함.
BIOS (Basic Input/Output System)
컴퓨터의 부팅 과정과 하드웨어 초기화를 담당하는 펌웨어로, 운영 체제 로딩 전에 실행됨.
RAID (Redundant Array of Independent Disks)
여러 개의 하드 디스크를 조합하여 데이터 중복 저장 및 성능 향상을 도모하는 기술.
